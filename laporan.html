Laporan Clustering

Pendahuluan

Laporan ini berisi langkah-langkah yang dilakukan untuk proses clustering pada data percakapan grup WhatsApp, mulai dari pembersihan data, vektorisasi, clustering menggunakan Metaflow, hingga analisis hasil.

Langkah-langkah yang Dilakukan

  1. Ekspor Data WhatsApp: Data dari beberapa grup WhatsApp diekspor dan digabungkan menjadi satu file (data_group.txt).
  2. Konversi ke CSV: File teks diolah di Excel dan disimpan dalam format CSV (data_group.csv).
  3. Kompresi File: File CSV dikompresi menjadi TAR (data_group.tar).
  4. Pembersihan Data: Karakter non-alfanumerik dihapus untuk mengurangi noise.
  5. Vektorisasi: Data teks diubah menjadi representasi numerik menggunakan TfidfVectorizer.
  6. Clustering: Data dikelompokkan menjadi 3, 4, dan 5 cluster menggunakan algoritma KMeans di Metaflow.
  7. Analisis Hasil: Kata-kata teratas dalam setiap cluster dianalisis untuk mengidentifikasi tema.

Analisis

Hasil clustering menunjukkan bahwa setiap cluster memiliki tema yang berbeda. Pada clustering 3, tema utama lebih umum, sedangkan pada clustering 4 dan 5, tema menjadi lebih terperinci dengan pembagian yang lebih spesifik. Namun, pada clustering 5, beberapa cluster memiliki overlap yang terlihat dari kemunculan kata-kata serupa.

Kesimpulan

Proses clustering berhasil mengelompokkan data teks menjadi beberapa kategori. Clustering dengan 3 atau 4 cluster memberikan hasil yang lebih terfokus, sementara clustering dengan 5 cluster mulai menunjukkan redundansi.

Dokumentasi File

  • data_group.txt: File teks hasil ekspor grup WhatsApp.
  • data_group.csv: File CSV hasil konversi dari teks.
  • data_group.tar: File TAR hasil kompresi.
  • scale_data.py: Script untuk vektorisasi data.
  • many_kmeans_flow.py: Script untuk clustering menggunakan Metaflow.
